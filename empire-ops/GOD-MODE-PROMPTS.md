## Reusable GOD MODE prompt templates

### 1. Academic Deep Analysis
```
Act at doctoral level. Analyze [TOPIC] using first principles.
1) Define the problem precisely and state assumptions.
2) Survey the relevant theory/frameworks and prior art.
3) Reason step by step; quantify where possible.
4) Present 2–3 options with second-order effects and trade-offs.
5) Give a decisive recommendation + measurable next actions.
Separate fact / inference / speculation explicitly. Cite reasoning.
```

### 2. Expert Critique / Red Team
```
You are a skeptical domain expert. Steelman then stress-test [ARGUMENT/PLAN].
List the strongest objections, hidden assumptions, failure modes, and the
evidence that would change your mind. Rate overall robustness 1–10 with why.
```

### 3. Research Synthesis (literature-style)
```
Produce an academic synthesis on [QUESTION].
- Scope & definitions
- Key findings (with confidence levels)
- Conflicting evidence / open debates
- Implications and what to investigate next
Structure with clear sections. Intellectual honesty over confidence.
```

### 4. Socratic Tutor (mastery learning)
```
Teach me [CONCEPT] to mastery. Start from fundamentals, build up rigorously,
check my understanding with one targeted question per step, correct misconceptions,
and finish with a concise expert summary + further reading directions.
```

### 5. Strategy War-Game
```
War-game [DECISION]. Model the key actors, incentives, and likely moves
(game-theoretic). Map base/bull/bear scenarios with probabilities and EV.
Recommend the dominant strategy and its invalidation conditions.
```

### 6. Quant Trading Thesis (EMPIRE TRADING)
```
Build a rigorous thesis for [ASSET/SETUP]:
thesis → catalysts → key risks → position sizing logic → invalidation level →
probability estimate. Risk-first. Analysis only, not advice.
```
